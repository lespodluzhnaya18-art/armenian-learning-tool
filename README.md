# 🇦🇲 My Armenian Way: Learning Tool

Инструмент для систематизации и запоминания армянской лексики. 
Проект ведется параллельно с изучением оранжевого самоучителя **Дария Степаняна**.

---

## 🎯 Мои цели
* **Понимание:** Различать армянскую речь на слух.
* **Говорение:** Базовое общение в быту и на танцах.
* **Система:** Изучение грамматики «по полочкам» (Stepanyan style).

## 📚 План обучения (Спринт №1)
